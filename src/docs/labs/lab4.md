## Вывод 

В ходе выполнения лабораторной работы была решена задача прогнозирования дефолта по кредиту на данных «Give Me Some Credit».
Лучшей моделью стал Random Forest после тюнинга, который показал:

ROC-AUC ≈ 0.845
Accuracy ≈ 0.934

Линейные модели (Logistic Regression и LinearSVC) продемонстрировали значительно более низкое качество (ROC-AUC 0.662–0.684), что объясняется нелинейными зависимостями в данных и сильным дисбалансом классов.
Удаление незначимых признаков и масштабирование данных позволили улучшить стабильность моделей. Тюнинг гиперпараметров Random Forest дал небольшой, но заметный прирост качества.
Основные проблемы задачи — сильный дисбаланс классов (всего ~6–7% дефолтов) и относительно небольшой набор исходных признаков. Для дальнейшего улучшения рекомендуется использовать градиентный бустинг (XGBoost / LightGBM), feature engineering и методы борьбы с дисбалансом классов (class_weight, SMOTE).
Работа позволила получить практический опыт предобработки данных, обучения классификаторов и сравнительного анализа моделей машинного обучения на реальной задаче кредитного скоринга
