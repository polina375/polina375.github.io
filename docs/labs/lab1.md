# Лабораторная работа 1: Создание статического сайта

## Цель работы
Освоить создание статического сайта с помощью MkDocs и его публикацию на GitHub Pages.

## Задание
1. Создать репозиторий на GitHub.
2. Установить MkDocs.
3. Настроить структуру и тему.
4. Опубликовать сайт.

## Код (команды)
```
mkdocs new source
cd source
mkdocs serve
mkdocs build -d ../docs    
```
