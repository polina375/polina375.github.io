# Цель работы

Изучение методов машинного обучения для решения задачи регрессии, а также построение модели для предсказания стоимости недвижимости. Освоение методов оценки качества моделей и базовой интеграции ML-модели в веб-сервис.

Целью данной задачи является прогнозирование стоимости домов в округе Кинг (штат Вашингтон, США) с помощью построения регрессионных моделей и их анализа. Набор данных состоит из цен на дома в округе Кинг, проданных в период с мая 2014 года по май 2015 года. Данные опубликованы в открытом доступе на платформе Kaggle.

## Задание
Загрузить и проанализировать датасет недвижимости

Выполнить предобработку данных

Обучить модели машинного обучения (Linear Regression, Random Forest)

Оценить качество моделей с помощью MAE, MSE, RMSE

Провести анализ важности признаков

Уменьшить количество признаков и сравнить качество моделей

Подобрать параметры Random Forest для улучшения качества

Построить более точную модель с использованием дополнительных алгоритмов

Интегрировать модель в веб-сервис (Flask)


## Вывод 
В ходе лабораторной работы были изучены методы построения регрессионных моделей для задачи предсказания стоимости недвижимости. Проведено сравнение различных алгоритмов машинного обучения, выполнена оптимизация модели и реализована её интеграция в веб-приложение. Полученные результаты показали, что ансамблевые методы (Random Forest, XGBoost) обеспечивают более высокую точность по сравнению с линейной регрессией.
